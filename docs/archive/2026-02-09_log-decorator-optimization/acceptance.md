# log_decorator 功能优化验收标准

## 1. 功能验收标准

### 1.1 优化点1：参数名称显示

**验收条件**：
- [ ] 位置参数显示实际参数名而非索引
- [ ] 关键字参数显示参数名
- [ ] 无法获取参数名时回退到索引显示
- [ ] self/cls 参数正确显示

**测试用例**：
```python
@log(print_args=True)
def process_intent(self, query: str, context: dict = None):
    pass

# 预期输出：
# ├─ process_intent - 【入参】
# │  ├─ self: <IntentService: 1 attrs>
# │  ├─ query: 危化品
# │  └─ context: None
```

**通过标准**：
- 所有参数都显示参数名
- 格式符合树形结构规范
- 无参数名时显示 `args[0]` 格式

---

### 1.2 优化点2：参数树形结构

**验收条件**：
- [ ] 参数作为函数节点的子节点显示
- [ ] 使用正确的树形连接符（├─、└─、│）
- [ ] 缩进层级与函数调用深度一致
- [ ] 多行参数保持对齐

**测试用例**：
```python
@log(print_args=True)
def outer():
    @log(print_args=True)
    def inner(x, y):
        pass
    inner(1, 2)

# 预期输出：
# ├─ outer - 【开始执行】
# │  ├─ 【入参】
# │  ├─ inner - 【开始执行】
# │  │  ├─ 【入参】
# │  │  │  ├─ x: 1
# │  │  │  └─ y: 2
```

**通过标准**：
- 树形结构清晰可读
- 层级关系正确
- 连接符使用规范

---

### 1.3 优化点3：默认不计算耗时

**验收条件**：
- [ ] `print_duration` 默认值为 `False`
- [ ] 不指定参数时不打印耗时
- [ ] 显式指定 `print_duration=True` 时打印耗时
- [ ] 耗时格式保持不变（毫秒，保留2位小数）

**测试用例**：
```python
# 场景1：默认不打印耗时
@log()
def func1():
    pass

# 预期输出：
# ├─ func1 - 【开始执行】
# └─ func1 - 【执行完成】  # 无耗时信息

# 场景2：显式打印耗时
@log(print_duration=True)
def func2():
    time.sleep(0.1)

# 预期输出：
# ├─ func2 - 【开始执行】
# └─ func2 - 【执行完成】耗时：100.23ms
```

**通过标准**：
- 默认行为符合预期
- 显式指定时正常工作
- 不影响其他功能

---

### 1.4 优化点4：删除 enable_mermaid 参数

**验收条件**：
- [ ] `log()` 函数签名中不存在 `enable_mermaid` 参数
- [ ] 保留 `force_mermaid` 参数
- [ ] 使用 `enable_mermaid` 参数时抛出明确错误
- [ ] 现有使用 `force_mermaid` 的代码不受影响

**测试用例**：
```python
# 场景1：使用已删除的参数（应报错）
@log(enable_mermaid=True)  # TypeError
def func1():
    pass

# 场景2：使用保留的参数（正常工作）
@log(is_entry=True, force_mermaid=True)
def func2():
    pass
```

**通过标准**：
- 函数签名正确
- 错误提示清晰
- 向后兼容性符合预期

---

### 1.5 优化点5：入口函数重复防止

**验收条件**：
- [ ] 日志文件名包含时间戳
- [ ] 格式为 `{func_name}_{timestamp}.log`
- [ ] 时间戳格式为 `YYYYMMDD_HHMMSS`
- [ ] 并发调用不会覆盖日志文件

**测试用例**：
```python
@log(is_entry=True)
def process_intent(query):
    pass

# 第一次调用
process_intent("危化品")
# 生成文件：logs/process_intent_20260209_173844.log

# 第二次调用（1秒后）
process_intent("易燃品")
# 生成文件：logs/process_intent_20260209_173845.log
```

**通过标准**：
- 文件名格式正确
- 多次调用生成不同文件
- 并发场景下无覆盖

## 2. 性能验收标准

### 2.1 性能影响

**验收条件**：
- [ ] 参数名称获取的性能开销 < 1ms
- [ ] 树形结构格式化的性能开销 < 0.5ms
- [ ] 整体日志记录性能下降 < 5%

**测试方法**：
```python
import time

# 测试1000次函数调用的平均耗时
def benchmark():
    @log(print_args=True, print_result=True)
    def test_func(a, b, c):
        return a + b + c

    start = time.time()
    for _ in range(1000):
        test_func(1, 2, 3)
    duration = time.time() - start

    # 平均每次调用耗时应 < 2ms
    assert duration / 1000 < 0.002
```

**通过标准**：
- 性能测试通过
- 无明显性能退化
- 内存使用无异常增长
