## 现有代码参考

### 接口定义 (src/rag_stream/src/routes/chat_routes.py)
```python
@router.post("/api/general", response_model=ChatResponse)
async def chat_general(request: ChatRequest, http_request: Request):
    intent_service = getattr(http_request.app.state, "intent_service", None)
    response = await handle_chat_general(
        request=request,
        intent_service=intent_service,
        chat_with_category=chat_with_category,
    )
    return response
```

### ChatRequest 模型
```python
class ChatRequest:
    question: str
    session_id: Optional[str] = None
    user_id: Optional[str] = None
    stream: bool = True
```

### 意图分类日志标记 (src/rag_stream/src/utils/intent_classifier.py)
- `classifier.attempt` - 开始分类
- `classifier.success` - 分类成功，包含 type_id 和 confidence
- `classifier.degraded_fallback` - 降级回退
- `classifier.low_confidence_fallback` - 低置信度回退

### 流式响应格式 (src/rag_stream/src/routes/chat_routes.py)
```python
async def dict_to_stream_generator(result_dict: dict):
    yield f"event: start\ndata: ...\n\n"
    yield f"id: {message_id}\nevent: message\ndata: ...\n\n"
    yield f"event: complete\ndata: ...\n\n"
    yield f"event: end\ndata: [DONE]\n\n"
```
