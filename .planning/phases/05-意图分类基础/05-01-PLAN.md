<task type="auto">
  <name>Task 2: 实现意图分类服务</name>
  <files>src/rag_stream/src/utils/intent_classifier.py</files>
  <action>
创建意图分类服务模块：

1. 定义分类结果数据类：
```python
@dataclass
class ClassificationResult:
    type_id: int  # 1/2/3
    confidence: float  # 0.0-1.0
    raw_response: str  # 原始响应
    degraded: bool  # 是否降级
```

2. 创建 IntentClassifier 类：
   - __init__: 接收 IntentClassificationConfig
   - _build_prompt: 生成分类 prompt（参考 CONTEXT.md）
   - _parse_classification_result: 解析 JSON 响应，提取 type_id 和 confidence
   - classify: 分类方法，返回 ClassificationResult
     - 如果 enabled=false，返回 degraded=True 的默认结果
     - 调用 QueryChat 的 chat.completions.create
     - 超时 3 秒（asyncio.wait_for）
     - 异常时返回 degraded=True 的默认结果

3. 分类 prompt 格式：
```
你是问题分类助手。请根据用户的问题，判断其属于以下哪种类型：

1 - 岱山-指令集
2 - 岱山-数据库问题
3 - 岱山-指令集-固定问题

只返回 JSON 格式：{"type": 数字编号, "confidence": 置信度(0.0-1.0)}，不要添加其他说明。

输入：{user_query}
输出：
```

4. 复用 marker 记录日志：
   - classifier.attempt: 分类尝试
   - classifier.success: 分类成功
   - classifier.error: 分类失败
   - classifier.degraded: 降级触发
  </action>
